Скрипт отключает защитник Windows (Windows Defender), без удаление файлов, отключение происходит за счет служб, драйверов и ключей реестра.
Так же отключает обновление Windows (Update Windows), иначе обновления могут ставиться не корректно, если планируйте ставить обновления, включайте Windows Defender.
